# TP1-Info2-JuanLisouski
Repositorio con el trabajo practico N°1 para informática 2, de Juan Lisouski, 2°11.
El código original provenía de un final resuelto de informática 1, el cual fue refactorizado, agregando funciones.c y funciones.h.
En funciones.c se encuentra el cuerpo de las funciones; en funciones.h se encuentran los prototipos de estas mismas funciones, junto a algunas definiciones y headers.
Se puede observar como main.c es facilmente legible, y el flujo del código es mucho mas fácil de seguir.
